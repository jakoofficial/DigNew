# Build b05 - 010626 Dev

## Note from developer
This build of the game is not a finalised version and not to be shared with others without permission from the developer.
This build is not representing of the final product and is subject to change.

## Changes
This dev build includes a complete redesign of the skill tree with a more clear and linear progression with newly configured prices and values(W.I.P).
This does mean a new save would be needed as older saves will not be compatable with the new version.

- As mentioned this new version includes a completly redesigned skill tree with more skills and new values in prices and stats given. The level unlocks have been moved to a more linear setup for ease of consistency.
- The artifacts have been modified with a new way of spawning so they no longer spawn in the top 2 rows and not in the bottom rows, this now works as intended.
- Level entry fees have been modified as well as these didnt match up with the new changes to the skill tree unlock prices.
- There have been a change to how the music and sound system works and is now moved to a new system to avoid loud scratching sounds when music starts after the game begins.
- New intro have been added at the start of the game, this only plays when the game opens first time and not when moving back the menu after the game has begun.
- Statistic information have been added to the bottom corner in the skill tree for easier understanding of the effects.

## Known issues
- Music can sometimes be muted after having it muted and then turned on again. Currently fixed by going to main menu or skill tree.
